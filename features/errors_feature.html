Error Feature

The Error Feature show all API Errors that have been thrown in the loaded events.



The Error Feature is the place to come if you are having problems and you think there might be an error with calling an API. Each Panel shows the name of the Error that was detected and the number of times that error occurs.

Clicking on a panel will display a Table View containing just the Events for the section of the panel you select.
