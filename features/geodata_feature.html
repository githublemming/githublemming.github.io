GeoData Feature

The GeoData Feature groups all Events together based on Source IP Address.



The GeoData Feature allow you to see where in the world API calls are being made from. Using a 3rd party library it attmpts to resolve IP addresses in the SoureIPAddress field to a City around the world. These events are then grouped by Panel showing the number of Events for that Area/ Note:The free 3rd library does not guarantee to be 100% accurate.

Located at the top right of the screen is the Map icon, clicking this will open your default web browser and show all events plotted on a Map. This is useful if you don't recognise a City or the IP address could not be resolved.

Clicking on a panel will display a Table View containing just the Events for the section of the panel you select.
