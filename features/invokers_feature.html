Invokers Feature

The Invokers Feature groups all Events together based on the IAM User and Role that called the API.



The Invokers Feature allows you too see what IAM Usert and Roles are make API calls. Each panel shows the name of the User or Role and the number of times they have invoked an API calls.

Note : This only includes your IAM objects, this screen does not show AWS Services that might be making calls on your behalf.

Clicking on a panel will display a Table View containing just the Events for the section of the panel you select.
