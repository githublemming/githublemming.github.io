Metrics Feature

The Metrics Feature plots all Events on a time series chart.



The Metrics Feature allows you too see the rate at which API calls are being made against the AWS Services.

You can zoom into the data by right-clicking on the chart and dragging to the right, to zoom out drag left.

Clicking on a node will display a Table View containing just the Events for the node you select.
