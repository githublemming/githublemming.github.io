Overview Feature

The Overview Feature provides a top level breakdown by AWS Service of the Events that have been loaded.



The Overview Feature is the main screen when Events are being loaded into the application. As new Events are loaded they are added to a panel that indicated which AWS Service the Event belongs too.

A panel provides the following information:

  • The name of the service
  • Who generated the Event.
  • The total number of Events loaded
  • The peak number of Events loaded per Second, Minute, Hour

A Panel is broken down into two areas:- Blue and Green.

The Blue section indicates calls that have been made by AWS as part of their management of your services. An example of this is Elastic Beanstalk checking the state of an ELB.
The Green section indicates calls that have been made by your Users/Groups or Roles.

Clicking on a panel will display a Table View containing just the Events for the section of the panel you select.
