Resources Feature

The Resources Feature shows user customisable Events for Resource manipulation.

On this screen you can view API calls that are relatd to Creating, Modifying or Deleting of AWS Resources. Like the Security Feature which API calls this feature will display is user configurable on the Preferences dialog.


Side Bar Chart

The Chart area of the side bar shows different information when using the Resources feature than on other screens. Depending on the Event you select in the main section the chart options change. For example if you select 'RunInstances' you can see what AMIs have been used, or what EC2 Instances Types where started.

Drill Down

Clicking on an API will display a Table View containing just the Events for the API you select.

Configuration

While it could be said that every AWS Account is probably using EC2s, the same could not be said for the rest of the services. In order that the Resources Feature shows only the information that is relevant to you the Events that are shown in this section are customisable. Configuration is done via the Resources Section of the Preferences dialog where you can add the API events that you are interested in.


CloudTrail Viewer comes with some recommended API Events but you can remove these if you wish and replace them with your own. Simply use the + and - buttons at the bottom of the dialog.
