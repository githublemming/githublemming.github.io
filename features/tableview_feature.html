TableView Feature

The TableView Feature is a main component of the CloudTrail Viewer.



The table view is a primary component of the application and it features heavily in the functionality of the majority of other features. The top feature Feature contains all load Events in chronological order the earliest being at the top of the table and the latest being at the bottom.

You can change the columns that you wish to display by right clicking on the table header and selecting the required columns from the list. You can also change the order of the columns.

Above the table is a filter box where you can filter the visible rows in the table. To the right of the filter bar are two buttons the first button exports the visible columns of the rows in CSV format, the second button sides the Side Bar.

Side Bar

Clicking a row in the table will display the side bar. The Side Bar consists of three parts:

  • Tree View : The CloudTrial Event selected will be displayed in a Tree Format
  • JSON View : The CloudTrial Event selected is shown in it's original JSON format. Additional fields used by the application such as City are also shown.
  • Chart View : The chart view shows a visual representation of the Top X loaded events as either a Bar or Pie chart. Using the menus you can change what the chart display.
