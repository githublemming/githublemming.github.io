CloudTrail Viewer

CloudTrail Viewer is an Open Source Java desktop application for reading and analysing the log files produced by the AWS CloudTrail service.

Use the application to monitor API usage, get more information about API Error, check for security issues.

Process your CloudTrail files to get complete visibility on what is happening in your AWS Account

Read Events

CloudTrail is a powerful feature, but it's no use if the files stay in S3 unused and unloved. Use CloudTrail Viewer to load the files from S3, or local filesystem, in order to view the information in an easy to read and understand format.

More Info

Analyse Usage

Once the files have been loaded use one of the many Features in CloudTrail Viewer to see the information in a way that works best for you. See Events broken down by User or Location and drill down to get more information.

More Info

Find Issues

Did an RDS disappear for no reason? Who's been spinning up all those EC2s? CloudTrail Viewer provides Features that organise the information loaded into useful sections to make it easy to monitor API activity.
